متجر إلكتروني عملي (واجهة أمامية فقط)

كيفية التشغيل:
1. افتح الملف `indx.html` في المتصفح (فقط فتح محلياً).
2. يتم تحميل المنتجات من `products.json`.
3. السلة مخزنة محلياً في `localStorage`.

ملاحظات:
- هذه نسخة تجريبية أمامية فقط ولا تحتوي على خادم دفع أو تخزين أو قواعد بيانات.
- يمكن توصيل واجهة الدفع بخدمة خلفية لاحقاً.

ملفات مهمة:
- `indx.html` — واجهة المتجر
- `styles.css` — أنماط التصميم
- `app.js` — منطق المتجر (تحميل المنتجات، السلة، الدفع الوهمي)
- `products.json` — بيانات المنتجات النموذجية
